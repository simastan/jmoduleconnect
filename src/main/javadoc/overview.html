jModuleConnect is an framework for communication and file management on modem modules.

The frameworks supports the AT command set for configuration and controlling modems and the OBEX protocol for access to flash filesystem. The adding of further protocol processsor is possible for devices the provides more then the AT command set on one port.

Optimized for Cinterion TC65 or EGS5.

Dependencys
RxTx rebundled Used for the access to the serial port.
SLF4J Used for the Logging.
