jModuleConnect is an framework for communication and file management on devices they are based on Cinterion TC65 or EGS5.

The frameworks supports the AT command set for configuration and controlling the modem and the OBEX protocol for access to flash filesystem. The adding of further protocol processsor is possible for device that contains more components like the modem mcu.

Dependencys
RxTx rebundled Used for the access to the serial port.
SLF4J Used for the Logging.
